#  SkinVerse – AI-Based Skincare System

SkinVerse is an AI-powered skincare assistant that analyzes skin conditions, detects issues, and recommends suitable products based on user data, ingredients, and preferences.

---

##  Overview

SkinVerse leverages AI and image processing to:
- Detect skin type and conditions
- Identify acne, spots, and skin issues
- Analyze skincare product ingredients
- Recommend personalized skincare routines

---

##  Key Features

###  Skin Analysis
- Detects skin type (oily, dry, combination)
- Identifies acne, pigmentation, and other issues
- Image-based analysis using AI

###  Product Evaluation
- Ingredient-based analysis
- Detects harmful or unsuitable ingredients
- Matches products with user skin profile

###  AI Recommendations
- Personalized skincare suggestions
- Considers:
  - Skin type
  - Allergens
  - Preferences
  - Environmental factors

###  Smart Input
- Supports:
  - Image upload
  - Voice input
  - Manual input

##  Tech Stack

### Frontend
- React.js
- Modern UI with responsive design

### Backend
- Node.js / Express
- Supabase (Database + Auth)

### AI / ML
- OpenAI API (recommendations)
- Image processing models (skin detection)

##  Installation & Setup

### 1. Clone the repository
cd skinverse
2. Install dependencies
Frontend
cd client
npm install
Backend
cd server
npm install
3. Environment Variables

Create a .env file:

SUPABASE_URL=your_url
SUPABASE_KEY=your_key
OPENAI_API_KEY=your_key

4. Run the project
Backend
npm start
Frontend
npm start
